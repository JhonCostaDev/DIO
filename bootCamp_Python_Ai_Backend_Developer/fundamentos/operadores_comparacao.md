# Conhecendo os operadores de comparação

São operadores utilizados para comparar dois valores

## Igualdade
saldo = 450
saque = 200

print(saldo == saque)
>>> True

## Diferença

print(saldo != saque)

## Maior que / maior ou igual

print(saldo >= saque)

##  Menor que / menor ou igual

print(saldo <= saque)
